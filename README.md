# Portfolio personal

Un proyecto de sitio web tipo portafolio personal.

## 📋 Descripción

Este repositorio aloja el código fuente para un portafolio web. La estructura del proyecto contiene los elementos esenciales de una página web frontal, destacando un archivo principal `index.html` y directorios organizados para estilos (`css`), fuentes tipográficas (`fonts`), imágenes (`img`) y otros recursos (`assets`).

El proyecto está diseñado y configurado para ser desplegado a través de la infraestructura de **Cloudflare** (como Cloudflare Pages o Workers), lo cual se evidencia por la presencia del archivo de configuración `wrangler.jsonc` y las contribuciones del bot automatizado de despliegue de Cloudflare.

### 💻 Tecnologías Usadas

El desarrollo de este portafolio se compone de los siguientes lenguajes web:

* **CSS:** 49.3%
* **HTML:** 37.1%
* **JavaScript:** 13.6%

### 📂 Estructura del Proyecto

Los directorios y archivos principales en la rama `main` incluyen:

* `assets/`: Carpeta con recursos adicionales.
* `css/`: Archivos de hojas de estilo.
* `fonts/`: Archivos de tipografías personalizadas.
* `img/`: Imágenes del sitio web.
* `index.html`: Archivo principal de la estructura de la página.
* `wrangler.jsonc`: Archivo de configuración para el entorno de Cloudflare.
* `favicon2.ico`: Icono del sitio para la pestaña del navegador.
* `.gitignore`: Reglas de exclusión para Git.
