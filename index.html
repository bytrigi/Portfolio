Windows Agenda for iOS 📅

Aplicación de escritorio (React/Electron) que provee una solución integral de gestión de tiempo con interfaz estilo macOS/iOS. Implementa una base de datos reactiva (Dexie.js) para persistencia local y uso sin conexión. La base técnica esta en el uso del protocolo CalDav en el cual se basa el calenadario de iOS, habilitando la sincronización nativa y bidireccional entre la aplicacion y el servicio de iCloud. Además, incluye un buscador global omnibar, diferentes opciones de navegación y las funcionalidades de hacer to-do lists con sus sublistas y la toma de notas.
